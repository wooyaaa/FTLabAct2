FTLabAct2:Pricing Table using FlexBox

Objective:

Pricing Table is used display the different pricing options for a product or service. It is designed to make it easy for users to compare features and costs between various plans or packages.

Student will create a Pricing Table using Flexbox.Also, this activity lets the students to:

  • Understand the basics of CSS Flexbox for layout design.
  • Apply Flexbox properties to achieve a structured pricing table layout.
  • Create a responsive and aesthetically pleasing Flexbox Pricing Table.

Instructions:

  1. This activity is should be done individually.
  2. An HTML File is already provided. You can download it here.
  3. Use an external stylesheet/CSS in styling the pricing table.
  4. Use display:flex to create a flexible layout for the flex container.
  5. The pricing tables must be positioned in center.
  6. The buttons should have a hover effect. Any transitions will do.
  7. Use media queries in order for the webpage to adapt to different screen sizes.
  8. Read the criteria for grading carefully.

Criteria for Grading:

  • Layout: 50%
    • Effective use of Flexbox properties.
    • Proper alignment and spacing of pricing items.
  • Responsiveness: 20%
    • Effective use of media queries.
    • The pricing table should be responsive and adapt to various screen sizes (pc and mobile).
  • Design: 15%
    • Consistent and visually appealing use of colors, fonts, and spacing.
    • Effective use of hover and transitions on buttons.
    • Clear and readable texts.
  • Usability: 15%
    • Buttons should be clearly visible and styled to encourage user interaction.
    • Overall user experience should be intuitive and pleasant.
